Chord Identification

Project Description

Chord Identification takes in a list of notes and returns potential chord names for that set of notes

This progrm works with live midi input from a midi controller to return the chord being played in real time or with manual input.

The program iterates through the input list making each note the root and calculating the intervals of the other notes. These intervals are then referenced against common chord interval compositions. Any matches are then put into a dictionary and shown as output.

Technical Sheet

These are the languages, topics, and frameworks explored in this project
  • Python
  • Music Theory
  • Midi (Musical Instrument Digital Interface)
  • Pygame (python package)

Background Story

This project is the beginning of a much larger idea that I have had for several years. I found that often when beginners start with music, they know what sounds good but not how to communicate why. The idea of the project is to help people better understand what they are playing by seeing the chords in real time. Another potential goal was to make some kind of visual display based on the notes played in relation to the key of the song (warm colors for resolved chords, harsher contrast for dissonance). There are definitely a few directions I want to explore with live music analysis so I am excited to expand this project.
