RateMyProfessor ASU Course Lookup

Project Description

This script takes in an ASU course (ie. 'CSE 110') and returns a summary of RateMyProfessor data for each teacher teaching that class

The program accesses a JSON version of the ASU course search website and gets a list of each teacher teaching a specific class. Those teachers are then looked up on RateMyProfessor.com and if a matching teacher is found the data is scraped and summarized in a text file as output.

Technical Sheet

These are the languages, topics, and frameworks explored in this project
  • Python
  • Webscraping
  • BeautifulSoup (python package)
  • JSON data

Background Story

This project was built purely for my own conveience. Some classes that I needed to sign up for had upwards of 15 teachers avaliable and it became laborious to check if each teacher was on RateMyProfessor and remember who had good reviews and who didnt. I figured that I could automate it with some webscraping (A skill I was interested in developing) and I found a way to do so. This project is one I'm looking forward to expanding because it could be genuinely useful to other students.
