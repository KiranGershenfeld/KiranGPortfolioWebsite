Markov Chains

Project Description

This script takes in sample text and immitates it using a markov chain algorithm.

The algorithm was written from scratch in python using tuplet dictionaries to catalouge each word in the sample text and what words potentially follow it. It then generates original text based on that data.

Background Story

This project was born from a conversation I was having with friends at the dinner table. They were talking about short fan fiction stories on the website WattPad and how they were all so similar it felt like they were written by robots. Naturally I wanted to see if I could make a robot to recreate these stories.

The first iteration of the project was a webscraper that went through wattpadd stories with specific tags and grabbed the text. Then I found the Markov Chain algorithm and set about making my own implementation. After some moderate (and hilarious) success I realized the project was worth expanding and I ended up presenting it as an Honors Contract for my CSE 110 class.
