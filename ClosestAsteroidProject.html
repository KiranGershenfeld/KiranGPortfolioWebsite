Closest Asteroid

Project Description

This website shows which small astral bodies are closest to Earth today, this month, and this year.

The website queries a public NASA API to request JSON data from the JPL Small Body database. This data is then processed and a the output is displayed on the screen.

Technical Sheet

These are the languages, topics, and frameworks explored in this project
  • HTML
  • CSS
  • Javascript
  • JQuery

Background Story

This project was a concentrated attempt to learn more about website development. While HTML and CSS are quite approachable, the nuances and best practices take time to learn and building this website helped me get a much better feel designing and implementing a website. It was also good practice with requesting JSON from APIs and processing data which is something that has come up in several projects of mine.
